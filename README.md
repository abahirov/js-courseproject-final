# Your Energy - Fitness Exercise App

Веб-застосунок для перегляду та відстеження вправ з категоризацією за м'язами, частинами тіла та обладнанням.

## Можливості

- 🏋️ Каталог вправ з фільтрацією
- ⭐ Система улюблених вправ
- 🔍 Пошук вправ за назвою
- 📊 Рейтинг вправ
- 💬 Цитата дня
- 📱 Адаптивний дизайн
- 🎨 Сучасний UI/UX

## Технології

- Vanilla JavaScript (ES6+)
- Vite
- CSS3 (Grid, Flexbox)
- REST API

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

## Білд

```bash
npm run build
```

## Деплой

```bash
npm run preview
```

Проєкт автоматично деплоїться на GitHub Pages через GitHub Actions.

## Структура

```
src/
├── css/           # Стилі
├── js/            # JavaScript модулі
│   ├── api/       # API запити
│   ├── handlers/  # Обробники подій
│   ├── storage/   # LocalStorage
│   ├── ui/        # UI компоненти
│   └── utils/     # Утиліти
├── partials/      # HTML компоненти
└── img/           # Зображення
```
